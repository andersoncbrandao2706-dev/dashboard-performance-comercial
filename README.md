# 📊 Dashboard de Performance Comercial

Projeto desenvolvido em **Power BI** com foco na análise de desempenho comercial, permitindo acompanhar indicadores estratégicos e apoiar a tomada de decisão por meio da análise de dados.

---

## 🎯 Objetivo do Projeto

Desenvolver um dashboard executivo para acompanhar a performance comercial, consolidando indicadores de vendas, metas e desempenho dos vendedores em uma interface intuitiva e interativa.

---

## 🚀 Principais Funcionalidades

- Visão Executiva dos principais indicadores comerciais.
- Comparativo entre faturamento e metas.
- Ranking de vendedores.
- Evolução das vendas ao longo do tempo.
- Análise da performance comercial.
- Navegação entre páginas.
- Filtros interativos por período, vendedor e grupo de produtos.

---

## 🛠 Tecnologias Utilizadas

- Power BI
- Power Query
- DAX
- Modelagem Dimensional (Star Schema)
- Power BI Service (Publicação)
